00267 /*
00268  * Current Maximum Length for Instant Messages
00269  *
00270  * This was found by experiment.
00271  *
00272  * The YMSG protocol allows a message of up to 948 bytes, but the official client
00273  * limits to 800 characters.  According to experiments I conducted, it seems that
00274  * the discrepancy is to allow some leeway for messages with mixed single- and
00275  * multi-byte characters, as I was able to send messages of 840 and 932 bytes
00276  * by using some multibyte characters (some random Chinese or Japanese characters,
00277  * to be precise). - rekkanoryo
00278  */
00279 #define YAHOO_MAX_MESSAGE_LENGTH_BYTES 948
00280 #define YAHOO_MAX_MESSAGE_LENGTH_CHARS 800
