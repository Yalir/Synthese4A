in-prpl function for joining a chat room. Doesn't require sticking goop into a hash table.

Parameters:
roomThe room to join. This MUST be normalized already.
serverThe server the room is on. This MUST be normalized already.
passwordThe password (if required) to join the room. May be NULL.
dataThe chat hash table. May be NULL (it will be generated for current core<>prpl API interface.)
