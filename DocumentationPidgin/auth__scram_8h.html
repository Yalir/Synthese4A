Calculates the proofs as described in Section 3 of the SASL-SCRAM I-D.

Parameters:
dataA JabberScramData structure. hash and auth_message must be set. client_proof and server_signature will be set as a result of this function.
saltThe salt (as specified by the server)
iterationsThe number of iterations to perform.
Returns:
TRUE if the proofs were successfully calculated. FALSE otherwise.

Implements the Hi() function as described in the SASL-SCRAM I-D.

Parameters:

strThe string to perform the PBKDF2 operation on.
saltThe salt.
iterationsThe number of iterations to perform.
